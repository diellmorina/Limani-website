# Foto Producent Limani Website

Ky është një website bazë për kompaninë Foto Producent Limani, një prodhues fotografish.

## Struktura e Projektit

- `index.html`: Faqja kryesore e website-it.
- `styles.css`: Stilet për dizajnin e website-it.
- `script.js`: JavaScript për funksionalitetin bazë (scrolling i butë).
- `images/`: Folder për imazhet (shtoni fotot tuaja këtu).

## Si të përdorni

1. Hapni `index.html` në një browser për të parë website-in.
2. Për të shtuar imazhe në portofol, vendosini në folderin `images/` dhe përditësoni referencat në `index.html`.

## Përshtatje

- Përditësoni tekstin dhe informacionet e kontaktit në `index.html`.
- Modifikoni stilet në `styles.css` për të ndryshuar pamjen.
- Shtoni funksionalitet shtesë në `script.js`.

## Teknologjitë e Përdorura

- HTML5
- CSS3
- JavaScript (ES6)
